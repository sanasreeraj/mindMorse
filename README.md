# MindMorse

**MindMorse** is an iOS app that enables silent, offline Morse code communication using touch gestures, haptic feedback, and sound — built to enhance accessibility for people with speech or hearing impairments.

---

## 📖 About

MindMorse bridges communication gaps by translating touch inputs into Morse code, which is then decoded into readable text. The app works entirely offline with no internet connection required, making it reliable in any environment. It combines tactile, auditory, and visual feedback to support a range of accessibility needs.

---

## ✨ Features

- **Touch-based Morse Input** — Tap to input dots and dashes using intuitive gesture controls
- **Real-time Decoding** — Instantly translates Morse code sequences into readable text
- **Haptic Feedback** — Vibration responses provide tactile confirmation for each input
- **Sound Feedback** — Audio tones accompany each dot and dash for auditory reinforcement
- **Silent Mode** — Operates without sound for discreet, quiet communication
- **Fully Offline** — No internet connection required; works anywhere, anytime
- **Accessibility-Focused** — Designed with communication accessibility as the core goal

## 🚀 Getting Started

### Prerequisites

- macOS with [Xcode](https://developer.apple.com/xcode/) installed
- iOS 15+ device or simulator

### Installation

1. **Clone the repository**
```bash
   git clone https://github.com/sanasreeraj/mindMorse.git
```

2. **Open in Xcode**
```bash
   cd mindMorse
   open MindMorse.xcodeproj
```

3. **Build & Run**
   - Select your target device or simulator
   - Press `Cmd + R` to build and run

---

## 📱 How It Works

1. **Tap** the screen to enter Morse code — short tap for a **dot (·)**, long tap for a **dash (−)**
2. MindMorse decodes the sequence in **real time** and displays the corresponding letter or word
3. **Haptic and/or audio feedback** confirms each input
4. Use in **silent mode** when discretion is needed
